# ToDo List v2
REST API example about a to-do list.

Version 2 includes pagination.

## Installation
- Start the API: `docker-compose up -d --build`
- Stop the API: `docker-compose down`

## Endpoints

|Method|Endpoint|Body params|
|-|-|-|
|GET|/todos||
|GET|/todos&offset=<start_item>||
|GET|/todos&size=<items_per_page>||
|GET|/todos/:id||
|POST|/todos|id, task, done|
|PUT|/todos/:id|task, done|
|DELETE|/todos/:id||

`offset` and `size` can be combined.

## Tools
Flask / Python
